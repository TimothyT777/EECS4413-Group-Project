## Installation and Setup
The following is required:
- Java
- Node.js
- Maven 3.x
- An IDE (Can use Eclipse or VSCode)
- Git

### Steps to Build and Run
1. **Clone the repository**
    ```bash
    git clone https://github.com/TimothyT777/EECS4413-Group-Project.git 
    cd project-PlaceHolderName
    ```
2. **Import into IDE**
    For Eclipse:
    - Open Eclipse → File → Import → Maven → Existing Maven Projects
    - Browse to the placeholderName folder (where pom.xml is)

    For VSCode
    - File → Open Folder
    - Browse to project-PlaceholderName
3. **Run the application**
    - Open PlaceholderApplication.java in src/main/java/com/example/placeholdername
    - Run as Java Application
    - Then, open your command line interface (Start menu → Run → cmd)
    - Browse to the frontend folder using your command line interface (cd ProjectLocation/project-PlaceholderName/placeholderName/frontend)
    - Start the react server in your command line interface using command: npm start
4. **Access in browser**
    - [http://localhost:3000](http://localhost:3000)
